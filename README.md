# 🎓 AI-Powered Homework Assistance System

An intelligent **AI-based educational web application** designed to help students with homework, grammar correction, study planning, quizzes, flashcards, voice interaction, and general knowledge learning.

This project is developed as a **B.Tech Mini Project** for exploring the use of **Artificial Intelligence in Education**.

---

## 📌 Project Overview

The **AI-Powered Homework Assistance System** is a smart learning platform that helps students:

- Understand concepts  
- Improve grammar  
- Plan studies  
- Revise using flashcards  
- Interact with AI  

### 🔥 Key Capabilities

- 🤖 AI-powered question answering  
- ✍️ Grammar correction  
- 📅 Study planner  
- ❓ Quiz system  
- 🧠 Flashcards  
- 🌍 General knowledge explorer  
- 🎤 Voice interaction  
- 📷 Image-based homework support (OCR)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/varsha-reddy18/homework_assistance.git
cd AI_HOMEWORK-ASSISTANCE
````

---

### 2️⃣ Backend Setup

```bash
cd backend
python -m venv venv
```

#### Activate (Windows)

```bash
venv\Scripts\activate
```

#### Install dependencies

```bash
pip install -r requirements.txt
```

#### Run server

```bash
python -m uvicorn main:app --reload
```

📍 Backend: [http://127.0.0.1:8000](http://127.0.0.1:8000)
📍 Docs: [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

---

### 3️⃣ Frontend Setup

Open `frontend/index.html` using:

* VS Code Live Server
  **OR**
* Browser
